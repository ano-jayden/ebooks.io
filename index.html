Welcome to My GitHub Pages!

This project is an ebook store management system built using SQLite and Python. Below is an overview of the project and its functionality:

  • Connect to and manage an SQLite database for storing ebook information.
  • CRUD operations to enter, update, delete, and search books.
  • Predefined dataset to get started quickly.

Project Repository

You can find the full source code and instructions in my GitHub repository:

GitHub Repository: ano-jayden/ebooks.io

Getting Started

  1. Clone the repository using git clone https://github.com/ano-jayden/ebooks.io.git.
  2. Navigate to the project directory and follow the README instructions.
  3. Run the Python script to interact with the ebook store database.

Database Initialization

The SQLite database is initialized with the following structure:

CREATE TABLE book (
    id INT PRIMARY KEY,
    title VARCHAR(255),
    author VARCHAR(255),
    qty INT
);

Features

The application supports the following features:

  • Adding new books.
  • Updating book details.
  • Deleting books from the database.
  • Searching for books by title or author.
